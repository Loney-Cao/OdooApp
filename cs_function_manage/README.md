# 单身狗管理系统

模型：

single.dog  
pair.history  

流程：

1.填写个人信息。  
2.查找符合条件。  
3.选择配对。  
4.发送配对申请。  
5.交往申请通过，正在交往。  
6.交往结果成功/失败。  

单身狗A，B，C，D。  
A填写资料。  
符合条件者：B，C，D。  
向B，C，D发送交往申请。  
谁先同意，就与谁交往。  
C率先点击了同意。  
A与C交往，A的正在交往对象变成C。  
配对成功，流程结束。写入配对历史。  
配对失败，写入配对历史。互相点评失败原因。C，D此时可以点击同意交往。
